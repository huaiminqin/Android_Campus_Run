# 跑步统计 App

基于 Android 的跑步统计软件，集成计步器、传感器、指南针、地图轨迹等功能。

## 功能模块

1. **跑步页面** - 实时显示距离、时长、配速、步数、卡路里，地图轨迹绘制，指南针方向
2. **历史记录** - 查看所有跑步记录列表
3. **数据统计** - 累计数据和本周数据统计
4. **工具页面** - 配速计算器、卡路里计算器、指南针

## 运行环境

- Android Studio 2021.1.1
- Gradle 7.1.2
- JDK 1.8
- minSdk 21, targetSdk 32

## 数据库配置

### MySQL 8.0

1. 执行 `database.sql` 创建数据库和表
2. 数据库连接信息：
   - 主机: localhost
   - 端口: 3306
   - 数据库: running_app
   - 用户: root
   - 密码: luo125314

### 数据表说明

- `users` - 用户信息表
- `running_records` - 跑步记录表
- `track_points` - 轨迹点表
- `daily_stats` - 每日统计表

## 运行步骤

1. 用 Android Studio 打开项目
2. 点击 `File` → `Sync Project with Gradle Files`
3. 连接设备或启动模拟器
4. 点击 `Run` 运行应用

## 权限说明

- 位置权限：GPS定位和轨迹记录
- 传感器权限：计步器和指南针
- 网络权限：地图加载

## 注意事项

- 计步功能需要设备支持计步传感器
- 地图功能需要网络连接
- 首次运行需要授予位置权限
